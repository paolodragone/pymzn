pymzn.solns2out¶

pymzn.solns2out(soln_stream, ozn_file, check_complete=False)¶

Wraps the solns2out utility, executes it on the solution stream, and then returns the output.

Parameters:
  • soln_stream (str) – The solution stream returned by the solver.
  • ozn_file (str) – The ozn file path produced by the mzn2fzn function.
  • check_complete (bool) – If True, a boolean value is returned, in addition to the solutions of the problem, indicating the completion status of the problem.
Returns:

Returns a list of solution strings. If check_complete=True the result is a tuple containing the solution list as first argument and a boolean value indicating the completion status of the problem as second argument.
Return type:

list or tuple
